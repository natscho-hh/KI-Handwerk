# KI Handwerk

Modulare SaaS-Plattform für Handwerks- und Dienstleistungsunternehmen in der DACH-Region.

## Vision

KI-gestützte Automatisierung für den Handwerksalltag – von Angeboten über Wartung bis zum Tagesgeschäft.

## Module

| Modul | Beschreibung | Status |
|-------|-------------|--------|
| **Angebotsmanager** | Angebote erstellen, verwalten, nachverfolgen | Prototyp |
| **Wartungsmanager** | Wartungsberichte, Termine, Erinnerungen | Live |
| **Tagesgeschäft** | CEO Tagesbericht, Morgen-Brief, Routinen | Live |

## Tech-Stack

- **Automatisierung:** n8n (Cloud)
- **Frontend:** HTML/CSS/JS, gehostet auf crashcage.de
- **Backend:** n8n Workflows + Google Sheets/Drive
- **KI:** Claude API, OpenAI API
- **Hosting:** Hetzner KVM, FTP auf crashcage.de
