# Tagesgeschäft

Modul für tägliche Routinen: Morgen-Brief, CEO Tagesbericht, Wetter-Updates.

## Funktionen

- Hamburger Morgen-Brief (Wetter, Termine, Aufgaben)
- CEO Tagesbericht (Zusammenfassung des Tages)
- Tagesabschluss (Gmail + Wetter)
- Regenwarnung für Roller-Fahrer
