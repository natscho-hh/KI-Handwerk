# Wartungsmanager

Modul für Wartungsberichte, Terminverwaltung und automatische Erinnerungen.

## Funktionen

- Wartungsberichte per Gmail empfangen und in Google Drive ablegen
- Automatische Ordnerstruktur pro Kunde/Objekt
- Erinnerungen für fällige Wartungen
- Google Sheets als Datenbasis
